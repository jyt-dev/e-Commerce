# E-commerce platform

# Overview

A scalable fullstack e-Commerce platfrom consists of many unique and important features that provide seamless user experience of selling and buying products online.

Feauters are :
- User Authentication
- Product Management
- Shopping Cart
- Order Processing
- Payment Gateway


# Tech Stack

- Tailwind CSS

## Backend

- Express.js
- MongoDB
- Cloudinary (cloud service)
